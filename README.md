# Falling Photos

## Usage

### HTML

```
<div id="lightbox">
  <img src="#">
</div>

<div id="falling-photos">
  <img src="imagesource.jpg">
  <img src="imagesource.jpg">
  <img src="imagesource.jpg">
</div>
```

### JavaScript

```import FallingPhotos from "fallingPhotos";

new FallingPhotos()
```

### CSS

Make sure to set a width and height for img elements inside of `#falling-photos`.
